## 三、商业模式

### 收入来源
```
免费层: 基础排盘 → 引流获客
黄金会员 ¥98/3月 (约 ¥33/月)
钻石会员 ¥1200/5年 (约 ¥20/月)
新版赛博大师 ¥8/周、¥88/年

+ 金币内购: ¥6-60 按需购买
+ 淘宝租号: ¥19.9/2月 (灰色市场)
```

### 估算：假设 10 万 DAU，5% 付费率 = 5000 付费用户
- 月收入: 5000 × ¥30 ≈ ¥15万/月
- 年收入: ~¥180万
