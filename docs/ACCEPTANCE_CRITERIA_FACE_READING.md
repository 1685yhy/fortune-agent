# CV 精确面相分析 — 验收标准

## 原则
> 不说「看起来像」，只说「测量出来是」。
> 每一个面相判断都必须有：精确数值 + 古籍出处 + 置信度。

---

## V1: 面部精确测量（MediaPipe 引擎）

### 验收标准
- [ ] V1.1 输入人脸照片 → 输出 468 个面部关键点坐标
- [ ] V1.2 计算 ≥ 15 项精确面部指标（见下方指标表）
- [ ] V1.3 支持侧光、暗光、正面、微侧脸（±15°）→ 误差 < 5%
- [ ] V1.4 多人脸照片自动选最大人脸
- [ ] V1.5 测量速度 < 200ms（单张照片）
- [ ] V1.6 非人脸照片返回友好提示（不崩溃）

### 测量指标表（15 项）
```
1.  三停比例：上停% / 中停% / 下停%
2.  脸宽 (mm)
3.  脸高 (mm)
4.  脸型分类：国字/圆/方/瓜子/长/三角/菱形/鹅蛋
5.  额头宽高比
6.  眼距/脸宽比
7.  眼型分类：桃花眼/丹凤眼/杏仁眼/细长眼/圆眼/下垂眼/三角眼
8.  眉型分类：柳叶眉/一字眉/剑眉/八字眉/新月眉/龙眉/扫帚眉
9.  鼻高/脸高比
10. 鼻型分类：悬胆鼻/蒜头鼻/鹰钩鼻/狮子鼻/剑锋鼻/直鼻/塌鼻
11. 嘴宽/脸宽比
12. 嘴型分类：樱桃嘴/方嘴/仰月嘴/覆船嘴/大嘴/小嘴
13. 耳型分类：金耳/木耳/水耳/火耳/土耳/招风耳/贴脑耳
14. 痣检测：位置 + 大小 (mm²)
15. 气色：LAB色彩空间 → 红润/偏黄/偏白/偏青/暗沉/正常
```

### 验收测试
```python
# 1. 精确度测试：用标准人脸数据集（LFW/ CelebA）100 张
#    → 脸型识别准确率 > 90%
#    → 五官比例测量误差 < 3%

# 2. 环境测试：同一人在不同光照/角度下
#    → 测量值波动 < 5%

# 3. 非人脸测试：发风景照/宠物照
#    → 返回「未检测到人脸，请发一张正面自拍」
```

## V2: 面相术语映射引擎

### 验收标准
- [ ] V2.1 将 15 项测量值映射为面相学术语
- [ ] V2.2 每种分类输出置信度（例如：悬胆鼻 87%，蒜头鼻 13%）
- [ ] V2.3 映射规则基于古典面相文献（《麻衣神相》比例标准）
- [ ] V2.4 同一人脸连续 3 次拍摄 → 分类结果一致（≥ 2/3 相同）
- [ ] V2.5 给出「最佳特征」和「需改善特征」各 ≥ 2 项

### 验收测试
```python
# 用 10 张标注好的面相照片
for photo in labeled_dataset:
    result = face_engine.analyze(photo)
    assert result.face_shape == labeled_face_shape  # 匹配率 > 85%
    assert 0 < result.forehead_ratio < 1.0
```

## V3: RAG 古籍验证 + LLM 报告

### 验收标准
- [ ] V3.1 每个面相判断附带 ≥ 1 条古籍引用
- [ ] V3.2 引用格式：「《麻衣神相·XX篇》：原文内容」
- [ ] V3.3 LLM 报告包含：总览 → 五官逐项 → 古籍依据 → 改善建议
- [ ] V3.4 报告长度 300-800 字（不啰嗦，有干货）
- [ ] V3.5 报告生成速度 < 5 秒（不含拍照上传）
- [ ] V3.6 同一人两张不同照片 → 报告主要内容一致（不矛盾）

---

## V4: 集成 + 回测

### 验收标准
- [ ] V4.1 微信中发送自拍 → 返回面相分析（通过 CoW）
- [ ] V4.2 API 端点：POST /api/face-reading（接受图片上传）
- [ ] V4.3 从网上收集 ≥ 50 张真实人脸照片做回测
- [ ] V4.4 回测指标：脸型准确率 > 85%，五官分类准确率 > 80%
- [ ] V4.5 与 3 位真实用户的面相结论对比（盲测）
- [ ] V4.6 8 场景回归测试全通过（新增面相不影响现有功能）

### 回测数据源
```
1. CelebA 数据集（公开人脸数据集，10万+标注图片）
2. 小红书/知乎 面相分析帖子（真实用户发图求分析）
3. 百度图片 "面相分析 案例"
4. 自己拍摄的测试照片（多角度、多光照条件）
```

## 超出预期的点

| 普通算命产品 | 我们的产品 |
|------|------|
| 「你的鼻子挺高的」 | 「鼻高/脸高比 0.62，属于『悬胆鼻』，置信度 92%。《麻衣神相》云：鼻耸天庭，四海驰名」 |
| 需要文字描述面相 | 发张自拍就行 |
| 分析结果莫名其妙 | 每个判断都有古籍出处 |
| 不同人分析差不多 | 精确测量保证千人千面 |
